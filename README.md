# Elecciones Scraper

This bot download info from `http://atlaselectoral.oep.org.bo/`


## Setup

```console

pip install -r requirements


```

Setup project dir
```
export PROJ_DIR=$PWD

```

## How to run:

```
python main.py

```

sudo service cron reload
